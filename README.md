## How I work each lesson

1. **Read** the lesson's `docs/en.md` myself.
2. **Note** the key ideas in my own words → `personal/notes/<phase>/<lesson>.md`.
3. **Code it from scratch** (no peeking at `code/`) → `personal/notebooks/<phase>/<lesson>.ipynb`.
4. **Compare** against the course's reference `code/` and against NumPy/PyTorch.
5. **Self-test** per phase.

> Rule: *"If I can't explain it in my own words, I didn't understand it."*
